# Contributing to Flutter Chess App

Thank you for your interest in contributing!  
This project is open-source and welcomes contributions that improve the app, fix bugs, or enhance the user experience.

## How to Contribute

1. **Fork the repository** and create a new branch for your work:
   ```bash
   git checkout -b feature/my-new-feature
Make your changes and test thoroughly.

Commit your changes with a clear message:

bash
Copy code
git commit -m "Add [feature/fix]: short description"
Push your branch to your fork and open a pull request.

Reporting Bugs
If you find a bug, please use the bug report template to submit it.
Include steps to reproduce, device info, and screenshots if possible.

Feature Requests
Have an idea for a new feature? Use the feature request template to describe it clearly.

Code Style
Follow Dart & Flutter best practices

Keep UI clean and consistent

Write readable, maintainable code

Security
If you discover a security vulnerability, do not open a public issue.
Please report it via the Security Policy.

Acknowledgements
Thanks to anyone who contributes! Your help improves this app for everyone.
